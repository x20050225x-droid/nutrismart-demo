# NutriSmart - AI 智慧健康飲食顧問

根據使用者的健康狀況與飲食習慣，提供個人化的營養分析與飲食建議。

## 功能特色

- **健康資料輸入**：輸入身高、體重、年齡、活動量等基本資訊
- **營養分析圖表**：使用 Chart.js 呈現營養素攝取比例
- **個人化建議**：根據使用者狀況生成飲食建議

## 技術架構

- **前端框架**：Tailwind CSS
- **圖表套件**：Chart.js
- **圖示庫**：Font Awesome
- **字型**：Noto Sans TC

## 使用方式

直接用瀏覽器開啟 `index.html` 即可使用，或透過 GitHub Pages 線上存取。
